---
title: Annotation panel options
comments: |
  This file is used in the following visualizations: heatmap, candlestick, state timeline, status history, time series
---

| Option                       | Description                                                                                                                               |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Enable multi-row annotations | Breaks each annotation data frame into a separate row in the visualization. Only visible in panels that display more than one annotation. |
| Enable annotation clustering | Combines high-density point annotations into region annotations.                                                                          |
| Hide lines and areas         | Controls display of annotation indicator lines, and shaded area for region annotations.                                                   |
