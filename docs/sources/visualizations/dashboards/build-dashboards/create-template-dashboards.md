# Create dashboards from templates

Grafana provides a variety of pre-built dashboard templates that you can use to quickly set up visualizations for your data.
These templates are designed to work with specific types data sources and metrics, making it easier to get started with monitoring and analysis.

## Use a template

To create a dashboard from a template, follow these steps:

1. Click **Dashboards** in the primary menu.
1. Click **New** and select **Dashboard from template** in the drop-down menu.
1. Select a template.

   The dashboard is created with a banner panel that indicates it's using sample data.

1. Click **Save dashboard** to make the dashboard it editable.
1. Update the data source for each panel and add queries.
1. When you've made all of your changes, remove the banner panel.
1. Click **Save dashboard**.
