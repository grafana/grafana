# Manage users

A _user_ is any individual who can sign in to Grafana. Each user belongs to one or more organizations and is assigned a role that determines their permissions.

- **[Manage users in an organization](./manage-org-users/)** — Invite users, assign roles, and remove users within an organization.
- **[Server user management](./server-user-management/)** — Manage users across the entire Grafana instance (server admin only).
