# Grafana Observability Datasource Frontend Library

> **@grafana/o11y-ds-frontend is currently in ALPHA**.

`@grafana/o11y-ds-frontend` is a collection of shared UI components and utilities used to build observability datasource plugins (Tempo, Jaeger, Pyroscope, etc.) in [Grafana](https://github.com/grafana/grafana). It provides settings components for trace correlations (TraceToLogs, TraceToMetrics, TraceToProfiles), NodeGraph, SpanBar, and Pyroscope profile type selection.

See [package source](https://github.com/grafana/grafana/tree/main/packages/grafana-o11y-ds-frontend) for more details.

## Installation

`yarn add @grafana/o11y-ds-frontend`

`npm install @grafana/o11y-ds-frontend`
