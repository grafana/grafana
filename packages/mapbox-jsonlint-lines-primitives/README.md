## Original description

A fork of the `lines-primitive` branch of [tmcw/jsonlint], which is a fork
of [zaach/jsonlint] that adds line number annotations to the parsed JSON.

Used by MapLibre GL style spec for providing helpful error messages when
validating GL style JSON documents.
