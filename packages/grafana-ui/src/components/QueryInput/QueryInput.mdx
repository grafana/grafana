# QueryInput

A controlled query input backed by CodeMirror. Use it for query text that may
span multiple lines and can be run with Shift+Enter or Ctrl+Enter.

### Usage

```tsx
import { QueryInput } from '@grafana/ui';

<QueryInput
  value={query}
  onChange={setQuery}
  onRunQuery={runQuery}
  placeholder="Enter a query (run with Shift+Enter)"
  aria-label="Query"
/>;
```
