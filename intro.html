Installation. Easy.

You've already got Logstash and Elasticsearch right?
Then you don't need nodejs, ruby, php, python or more than a couple minutes. You do need a webserver and a good browser.

1. Download @github

Or head over to the Github project page

Github Project Page »

2. Setup one thing

You've made it this far. I'm going to bet you know how to:

  1. Extract your archive
  2. Open config.js in your favorite editor
  3. Set elasticsearch: 'http://localhost:9200', to your ElasticSearch server
  4. Copy The entire directory to your webserver
Heads Up Don't miss the comma at the end of line in step 3!

3. Awesome that was easy

  1. Point your browser at your shiney new installation
  2. The default dashboard has a gist link you can load to get a logstash dashboard view

For more information, including a short introduction to Lucene search syntax, check out the about section of this page.

A Note

Kibana 3 talks directly to Elasticsearch from the browser. This means that your browser communicates directly with Elasticsearch, not via an intermediary. You may wish to configure a reverse proxy to restrict access to Elasticsearch
