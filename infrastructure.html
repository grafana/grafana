Welcome to Kibana 3

Kibana is an open source (Apache Licensed), browser based analytics and search interface to Logstash and other timestamped data sets stored in ElasticSearch. With those in place Kibana is a snap to setup and start using (seriously). Kibana strives to be easy to get started with, while also being flexible and powerful, just like Logstash and Elasticsearch. Check it out ...

Log Search. Born for it.

Kibana was created to search logs and machine generated data. It plays extra nice with Logstash, using Logstash's time stamped indices to query Elasticsearch in the most efficient way.

Find 'Mozilla' in any field of any event, even if you never used a Logstash filter to define a single field

Mozilla

If you've defined a field called agent to store user agent strings:

agent:"Mozilla Firefox"

We can also search numerical ranges. Lets find every zip file downloaded larger than 300 bytes

extension:zip AND bytes:[300 TO *]

Dashboarding. Done Right.

Kibana 3 ships with a great interface to Logstash, but it doesn't stop there. Design your own custom dashboards with configurable panels that can be grouped by role. The possibilities are endless and we're adding fun new features, panels and visualizations all the time. Charts and panels are fully interactive with features like drill down, range selection and customization. They're also totally sharable and saveable.

  • Multiple concurrent search inputs
  • Hightlight to drill down bar charts
  • Line charts, stacked, unstacked, filled or unfilled, with or without points
  • Pie and donut charts that compare top terms or the results of multiple queries
  • Maps of the World, US and Europe more coming soon
  • Flexible date and time selection with configurable auto refresh interfaces
  • And growing!

Be the Builder. No code required.

You're not here to write code. You're here to get things done. Kibana 3 allows you to build custom dashboards right in your browser without writing a line of code. Dashboards are built on a grid system with fixed height rows and variable width panels that are given a 'span' of 1 through 12 depending on how much of the screen you want them to use. And the best part: Its all point + click + awesomeness.

By now you're thinking: Oh great, another black box that I can't integrate with my configuration management system. No way homefry. Under all the slick, colorful, Bootstrap based control, we're cooking up fresh JSON, and you know you can build that programatically. Drop you totally awesome puppet/chef/cfengine generated dashboard in dashboards/myawesomeness and point your browser at http://localhost:8000/index.html#/dashboard/file/myawesomeness. Who's awesome? You're awesome.
